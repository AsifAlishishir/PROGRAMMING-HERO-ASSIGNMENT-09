# üêæ PETPAWS: Pets in Care üè°

## A Modern Pet Adoption and Care Platform

**Developed by: Asif**

PETPAWS is a contemporary, full-featured web application designed to connect loving homes with pets in need of care and adoption. Built using React and Vite, the platform offers a seamless and responsive user experience.

---

## ‚ú® Features

* **User-Friendly Interface:** Built with **React** and styled using **Tailwind CSS** with **DaisyUI** for a modern, component-driven look.
* **Smooth Navigation:** Utilizes `react-router` for fast, client-side routing.
* **Animations:** Incorporates the `motion` library for engaging and smooth UI transitions.
* **Authentication & Data:** Integrated with **Firebase** for secure user authentication and backend data management.
* **Responsive Sliders:** Uses the `swiper` library for mobile-friendly image carousels and showcases.
* **Notifications:** Provides professional user feedback using `react-toastify`.
* **Iconography:** Rich set of icons powered by `react-icons`.

## üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have **Node.js** installed on your system.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [YOUR_REPO_URL]
    cd warmpaws_by_asif
    ```

2.  **Install NPM packages:**
    ```bash
    npm install
    ```

3.  **Set up Firebase:**
    * Create a Firebase project.
    * Add your project configuration environment variables (e.g., in a `.env` file).

### Running the Project

To start the local development server:

```bash
npm run dev
````

The application will typically be available at `http://localhost:5173/`.

### Building for Production

To create an optimized build of the application:

```bash
npm run build
```

The production files will be output to the `dist` directory.
